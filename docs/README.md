## Installing

A step by step series of examples that tell you how to get a copy running.

Clone the repo

```
git clone https://github.com/bierteam/Pils.git
```

Go to the directory

```
cd Pils
```

Install project

```
npm install
```

Copy and fill config file

```
cp config.example.js config.js
```
Run server
```
node server.js
```
